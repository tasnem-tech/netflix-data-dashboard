# 🌍 Overview

**Netflix Analytics Dashboard** is an interactive data visualization project built with **Python**, **Pandas**, and **Streamlit**.

The dashboard transforms raw Netflix data into meaningful insights, helping users explore trends, content distribution, ratings, and global availability through interactive charts and key performance indicators.

---

# ✨ Features

| Feature | Description |
|----------|-------------|
| 🎬 Movies vs TV Shows | Compare content distribution |
| 🌍 Country Analysis | Explore content availability by country |
| ⭐ Ratings Analysis | Analyze age ratings and certifications |
| 📈 Content Trends | Discover release year trends |

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/tasnem-tech/netflix-data-analysis-dashboard.git
```

Navigate to the project

```bash
cd netflix-data-analysis-dashboard
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
