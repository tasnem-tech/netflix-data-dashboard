## 📌 Overview

This project is an interactive Netflix Data Analysis Dashboard built using Python and Streamlit.

The dashboard analyzes:
- Movies vs TV Shows
- Country-wise content
- Ratings distribution
- Netflix trends

---

## 🚀 Features

✅ Interactive Dashboard  
✅ Data Visualization  
✅ Country Analysis  
✅ Content Comparison  
✅ Python Data Analytics  

---

## 🛠 Technologies Used

- Python
- Pandas
- Matplotlib
- Streamlit
